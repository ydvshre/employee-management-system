# Employee Management System

A full-stack Employee Management System built using **Spring Boot, Thymeleaf, and MySQL**.  
This application allows administrators to manage employee records efficiently through a clean web interface.

---

## Features

- Add new employees
- Edit employee details
- Delete employees
- View all employees
- Search employees by ID
- Generate employee reports
- Admin dashboard interface

---

## Tech Stack

Backend
- Java
- Spring Boot

Frontend
- HTML
- CSS
- Thymeleaf

Database
- MySQL

Tools
- Maven
- Git
- GitHub
